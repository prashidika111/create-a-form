**Create-a-form**
A simple practice project to have hands-on with dynamic rendering, form architectures, state-driven UIs.  
I tried using form schema library as well just so there is no need to keep repeating the form elements framework over and over.  The UI is pretty much simple, equipped with Tailwind CSS. The user can build their own forms dynamically with the provided field options like text, select, checkboxes as well as file uploads (from one to multiple files) as well as preview the form being created in realtime and generate the corresponding JSON representation.
